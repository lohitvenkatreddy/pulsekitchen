🔥 Core Priority Algorithm

Priority Score = 0.4 × urgency + 0.35 × distance + 0.25 × waiting_time

Urgency (40%)

Factors in VIP status, order type (express/hospital/student), and peak hours.

Distance (35%)

Calculates Haversine distance; shorter distances receive higher scores.

Waiting Time (25%)

Prevents starvation by dynamically boosting the priority of older orders.
